# Get This Work Done

Small Python setup and mini-project exercises.

## Setup

1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Put `FAKE_API_KEY=abc123` in a local `.env` file.

## Scripts

- `env_reader.py` loads `.env` and prints `FAKE_API_KEY`.
- `chat_history.py` runs a simple terminal chat loop with `history` and `quit`.
